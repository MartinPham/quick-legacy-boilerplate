# Quick Legacy Boilerplate

Quick boilerplate for legacy projects

- ES
- SCSS
- SSI
- BrowserSync

***Install dependencies***

```
	yarn
```

***Run Gulp***

```
	gulp
```

***Inline CSS***

```
	gulp --inline-css
```

***Build HTML***

```
	gulp --build-html
```

***Build Inline Email***

```
	gulp --inline-css --build-html
```
